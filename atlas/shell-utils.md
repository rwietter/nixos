# Shell Utils

### Brace Expansion

```bash
mv /path/to/dir/{flake.nix, flake.lock, .envrc} /path/to/dir
```

### Abbreviations

```bash
git-
nx-
```

### Forgit

```bash
forgit-

Ctrl + a # git-forgit add
Ctrl + d # git-forgit diff
```

## Trap

```bash
# trap <task> <event>
trap "rm test.nix" EXIT
```
